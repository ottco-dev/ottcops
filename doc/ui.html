Drag & Drop

  • Drop files onto the dropzone to populate the file input.
  • Hovering highlights the dropzone; leaving removes the highlight.

Image preview & zoom

  • Thumbnails appear for each selected file.
  • Click a thumbnail to open the preview modal; zoom via the slider.

Theme switch

  • Use the theme toggle to flip between light/dark.
  • The choice is stored in localStorage and restored on load.

JSON fullscreen

  • Click “Fullscreen” near the JSON output to open an overlay.
  • Close via the top-right button to return to the normal layout.

Code reference & adaptation tips

  • Drag & drop: wireDropzone() and renderImagePreviews() in static/js/app.js are standalone; call them with your own input/preview selectors to reuse the behavior.
  • Theme switch: toggleTheme() flips theme-dark/theme-light classes on body and saves to localStorage. Drop the toggle if you want a fixed style.
  • JSON fullscreen: openJsonFullscreen() renders an overlay for any JSON string. You can point it at another textarea or API response for quick inspections.
// Reuse dropzone on another page
wireDropzone({
  dropzoneId: 'myDrop',
  fileInputId: 'myFileInput',
  onFiles: files => renderImagePreviews(files, document.getElementById('myPreview'))
});

Tip: every widget is optional—include only the HTML blocks you need and the JS will gracefully ignore missing elements.
