Teachable Machine (easy mode)

  1. Train an image classifier in Teachable Machine.
  2. Export as TensorFlow (TFJS) or Keras; ZIP with metadata.json/model.json/weights.bin or keras_model.h5 + labels.txt.
  3. Upload via the config page; the server converts TFJS if needed and registers labels.
  4. Select the model in the analyzer or set it as default.

Pro mode: Label Studio + YOLO

  1. Label images in Label Studio (bounding boxes or segments).
  2. Export COCO and train a detector (YOLOv8/YOLOv10) on GPU.
  3. Convert the trained model to a TensorFlow SavedModel or .h5/.keras.
  4. Package the export with a labels.txt describing class order, then upload.

Keep class names concise; they appear in JSON outputs and summaries.

Model registry

  • Uploads land in TM-models/ with metadata stored in registry.json.
  • The active default is stored in app-settings.json and can be changed via the config console.
  • ML-only mode warms up the selected bundle during startup to reduce first-run latency.
