Export-Funktionen

Reports können als JSON, PDF oder Share-Link weitergegeben werden.

Zurück Home-Automation

JSON-Download

  • Button „JSON herunterladen“ anklicken.
  • Das UI generiert aus dem aktuellen Ergebnis eine Datei opencore-report.json.
  • Bei Batch-Analysen enthält sie Summary und alle Items.

PDF-Report

  • Button „PDF-Report“ erstellt ein PDF direkt im Browser.
  • Inhalt: Datum/Uhrzeit, ausgewähltes Modell, Summary, auf Wunsch verkürzte Einzelresultate.
  • Die Generierung nutzt jsPDF clientseitig, es verlassen keine Daten den Browser.

Share-Link

  1. „Share-Link erzeugen“ klicken.
  2. Das Frontend sendet den Report an /api/opencore/share.
  3. Die Antwort enthält eine URL /share/<id> und einen Copy-Button.
  4. Der Viewer lädt die Daten serverseitig und zeigt sie in einer lesbaren Karte an.
POST /api/opencore/share
{
  "payload": { ... Analyse-JSON ... }
}

Antwort:
{
  "share_id": "abc123", "url": "/share/abc123"
}

Share-Daten liegen serverseitig als JSON-Dateien und können bei Bedarf manuell gelöscht werden.
