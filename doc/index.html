Overview

The Analyzer pairs Teachable Machine models with LLM outputs and returns clean JSON. The front end and backend stay classic: FastAPI + HTML/JS/CSS without extra frameworks. Every feature can run independently and is documented here.

Highlights

  • Prompt Templates: Auto-prompts plus per-browser custom presets.
  • Batch & ML-only: Multi-image uploads and pure TM inference without LLMs.
  • Debug Panel: Request IDs, model versions, and timing breakdowns.
  • API Token Mode: Custom base URL + token for enterprise targets.
  • UI Enhancements: Stream dashboard, theme switch, JSON fullscreen, drag & drop.
  • Export: JSON download, PDF report, and share links with a viewer.
  • Video & Streams: Scheduler for snapshot/RTSP cameras with batch automation.
  • ML Models: Tutorials for Teachable Machine (easy) and Label Studio + YOLO (pro).
  • Custom Modules: How to attach your own backend modules and front-end hooks.
  • Raspberry Pi Guides: Mounting in grow tents, camera setup, edge scripting.
  • Home Automation: Examples for Home Assistant, Node-RED, and Python scripts.
  • LLM Switchboard: `/api/settings/llm` persists provider/base URL/model/system prompt for Analyzer & OTTO.
